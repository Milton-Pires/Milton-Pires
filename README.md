### 📌 Projeto em destaque

#### [iSpec – Sistema de Inspeção de Equipamentos de Incêndio](https://github.com/Milton-Pires/ISpec-WEB)

> Aplicação web full-stack para gestão e inspeção de extintores, alarmes e hidrantes, desenvolvida como TCC.

- 🔐 Autenticação JWT + Google OAuth2 com controle de acesso por perfil (Admin / Fiscal / Técnico)
- 📊 Geração de relatórios em PDF e Excel
- 📅 Agenda visual de inspeções com categorias por cor
- 🔔 Sistema de avisos automáticos para equipamentos vencidos
- 📱 API REST consumida por app Android complementar
